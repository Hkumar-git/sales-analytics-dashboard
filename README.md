# 📊 Sales Analytics Dashboard

## 📌 Project Overview

This project analyzes sales data using Excel, SQL, and Power BI to generate meaningful insights.

## 🛠 Tools Used

* Excel (Data Cleaning)
* SQL (Data Analysis)
* Power BI (Dashboard Visualization)

## 📊 Key Features

* Total Sales KPI
* Region-wise sales analysis
* Category-wise distribution
* Monthly trends
* Interactive filters

## 📈 Insights

* Central region has highest sales
* Technology category performs best
* Sales vary across regions
